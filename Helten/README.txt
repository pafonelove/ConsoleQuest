/// Description ///
Helten is the text-oriented console RPG which inspired by roguelike, fantasy, text, quest and RPG games.
The game is written in C# programming language. My close people Asurisss and GronKansla are helping me develop this game.

/// Control ///
Before starting the game, open it in full screen mode (Alt + Enter).
Contol is carried out by entering a specific number and pressing the Enter key. If no number is shown on the screen, you must press the Enter key to continue.

/// Bugs, errors ///
The game is currently under active development and has demo version 0.0.1 (alpha) for Windows OS. A Linux version is on the way.
If you find any bugs, please report them to one of the following resources:
- Game Jolt game page;
- e-mail (indicated in contacts);
- write to Telegram (id is indicated in contacts).
Most of the functionality of the game is implemented thanks to the work with the files contained in the "audio", "txt" folders. For the correct operation of the game, I do not recommend modifying these files in any way.
If you decide to change the files for the sake of interest or the desire to break the game for the sake of finding bugs and errors, then do not forget to make a backup copy of the original files =)

/// Описание ///
Helten — это текстовая консольная ролевая игра, вдохновленная рогаликами, фэнтези, текстовыми, квестовыми и ролевыми играми.
Игра написана на языке программирования С#. В разработке этой игры мне помогают мои близкие люди Asurisss и GronKansla.

/// Управление ///
Перед началом игры откройте ее в режиме полного экрана (Alt + Enter).
Управление осуществляется вводом определенного числа и нажатием клавиши Enter. Если на экране не отображается номер, для продолжения необходимо нажать клавишу Enter.

/// Баги, ошибки ///
В настоящее время игра находится в активной разработке и имеет демо-версию 0.0.1 (альфа) на ОС Windows. Версия для Linux - в планах.
Если вы обнаружите какие-либо ошибки, сообщите о них на один из следующих ресурсов:
- страница игры Game Jolt;
- электронная почта (указана в контактах);
- написать в Telegram (id указан в контактах).
Большая часть функционала игры реализована благодаря работе с файлами, которые содержатся в папках "audio", "txt". В целях корректной работы игры я не рекомендую каким-либо образом изменять эти файлы.
Если же вы решите изменять файлы в целях интереса или желания поломать игру ради нахождения багов и ошибок, то не забудьте сделать резервную копию оригинальных файлов =)
